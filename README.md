## 📌 Sobre o Projeto
Este repositório nasceu como um projeto de modelagem rigorosa de banco de dados para uma cafeteria. Na versão 2.0, o projeto evoluiu para uma **Aplicação Full-Stack**, integrando o banco de dados SQL Server a uma API em Node.js e um Dashboard administrativo moderno com foco em UX/UI.

---

## 🎨 Interface V2.0 (Full-Stack)
A nova camada visual foi desenvolvida seguindo tendências modernas de design para administradores:
* **Tema Purple Dracula:** Estética Dark com acentos roxos e marrom café.
* **Glassmorphism:** Uso de efeitos de transparência e desfoque (*backdrop-filter*) nos containers.
* **Monitor de Estoque (Gauge):** Um gráfico de velocímetro animado que reflete o nível de insumos no banco de dados.
* **Terminal de Auditoria:** Feed em tempo real que exibe os logs gerados pelas Triggers do SQL Server.

---

## 🛠️ Tecnologias Utilizadas
* **Banco de Dados:** Microsoft SQL Server (T-SQL).
* **Backend:** Node.js, Express, Driver MSSQL.
* **Frontend:** HTML5, CSS3 (Grid/Flexbox), JavaScript (Vanilla), Particles.js.
* **Modelagem:** dbdiagram.io e GitHub para versionamento.
